REMOTE SUPPORT APPOINTMENTS

here we'll assist you to connect with a certified remote support agent to resolve your IT issues. if you need to make a new appointment, you can click here, or just email us at info@example.com.

if it's already time for your appointment, you should receive a phone call as soon as the support agent is ready to help you. In preparation to receive one-on-one IT and network support, you should do the following:

  • Download and install the TeamViewer QuickSupport app here:
    • Windows
    • Mac
    • Linux
  • Get the following ready for your support agent:
    • A cabled internet connection to the computer or network to be serviced, such as an Ethernet, Coax, or Fiber uplink
    • An administrator username and password for the machine or network to be serviced (we recommend a temporary admin account and/or password for the explicit purpose of your appointment)
    • Your TeamViewer Partner ID code
  • If you are not able to get these things ready, your remote support agent will be happy to assist you.
